# ARTIFICE_CORE V1.0

A neuromorphic NPC agent built on a Leaky Integrate-and-Fire (LIF) spiking neural network.

## Features
- **Neural Substrate:** 30-neuron SNN with Input, Salience, and Output layers.
- **STDP Learning:** Synaptic weights adjust in real-time based on the timing of spikes ("Neurons that fire together, wire together").
- **Neurochemical Engine:** Simulates Dopamine (reward/curiosity) and Cortisol (stress) which affect neural thresholds and decay rates.
- **Embodied Visuals:** A Three.js icosahedron that pulses, glows, and moves based on internal neural activity.
- **Semantic Transduction:** Maps natural language to specific voltage injections in the neural layers.

## Usage
1. Click **Initialize Consciousness** to start the engine.
2. Communicate via the terminal to stimulate the sensory neurons.
3. Use **Extract Consciousness** to save the current state of the brain's "Connectome" to a JSON file.
